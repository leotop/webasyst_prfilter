# Модуль дополнительные характеристики
Стандартный фильтр продуктов в категории довольно скудный, и покупателю он практически не помогает отыскать нужные ему товары. За идею данного модуля были взяты популярные сайты яндекс.маркет и m.ua, которые действительно удобные, что даже не "продвинутый" пользователь сможет найти товар тот, который ему нужен.

	
<h2>Что реализовано в данной модуле?</h2>
	- Модуль дает возможность выбора нескольких параметров из одной характеристики.
	- Характеристики могут быть обычными, слайдерами, и с единичными значениями.
	- Для обычных характеристик можно указать тип: checkbox, radio или select.
	- Для обычных и единичных характеристик можно добавить изображение.
	- Модуль ищет продукты в подкатегориях и в родительских категориях.
	- Есть возможность указать рекомендуемые параметры, которые отображаются 
	  в первую очередь. И соответственно есть возможность показать все параметры характеристики.
	- Для категории фильтров и для самих параметров можно указать описание и заголовок описания.
	- Для параметров характеристики можно добавить исключенные фильтры, которые 
	  блокируют другие характеристики.
	- Полная мультиязычность модуля.
	- Есть возможность создавать категории характеристик, благодаря которым упрощается поиск 
	  нужной характеристики.
	- Благодаря редактору шаблонов, Вы можете создавать уникальные наборы характеристик 
	  для каждой отдельной категории или для всех сразу.
	- Есть возможность размещать блоки характеристик для любой страницы магазина.
	- Импорт и экспорт характеристик в .csv файл значительно упростит работу.
	- Добавлена страница http://ваш_магазин/psearch, на которой можно подобрать 
	  характеристики для любой категории.


<h2>Установка</h2>

	Установка модуля полностью автоматизирована и состоит она всего из нескольких пунктов.
	Внимание! В ходе установки, в файле /published/SC/html/scripts/index.php отключается 
	файл функций "option_functions.php".	Все функции в этом файле добавлены в конец 
	файла модуля /published/SC/html/scripts/core_functions/prfilter_functions.php, и 
	некоторые из них модифицированы.

<b>Шаг 1</b>

	Залить файлы из архива в папку с webAsyst shop-script.

<b>Шаг 2</b>
	
	Авторизуйтесь в админ-панели, затем перейдите по адресу http://домен/install.php.
	Ознакомьтесь с условиями предоставления скриптов нажмите "Установить".

<b>Шаг 3</b>

	Скрипт автоматически проверит наличие файлов модуля. При возникновении ошибки, 
	убедитесь что файлы находятся в нужной папке. При возникновении	ошибки 
	"Неверный размер" файла убидитесь в корректной загрузке файла. Возможно 
	он был поврежден в процессе загрузки на сервер.	Для продолжения нажмите "Продолжить".

<b>Шаг 4</b>

	Скрипт автоматически внесет изменения в файлы webAsyst. При возникновении 
	проблемы, установите права записи на указанные файлы 777, затем измените 
	их на первоначальные права. Для продолжения нажмите "Продолжить".

<b>Шаг 5</b>

	Скрипт автоматически добавит записи в базу данных. Если часть записей уже 
	находится в базе данных, то они будут удалены и перезаписаны  заново.
	Для продолжения нажмите "Закончить установку".

<b>Шаг 6</b>

	Зайдите в админ панель в раздел "Дизайн" - "Языки и перевод" 
	- Выберите ваш язык и нажмите "Редактировать перевод".	
	- Опуститесь в самый низ и нажмите "сохранить". 

<b>Шаг 7</b>

	Зайдите в админ панель в раздел "Команда" и проставьте права доступа к модулю. 


<b>Шаг 8</b>

	В редакторе дизайна, в head вставте следующую строчку
    <script type="text/javascript" src="http://ajax.googleapis.com/ajax/libs/jquery/1.10.2/jquery.min.js"></script>
